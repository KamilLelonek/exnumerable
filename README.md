  1. Add exnumerable to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:exnumerable, "~> 0.0.1"}]
end
```

  2. Ensure exnumerable is started before your application:

```elixir
def application do
  [applications: [:exnumerable]]
end
```
